# 타임헤어 미용실 관리 앱 개발 문서

## 1. 프로젝트 개요

### 1.1 프로젝트 목적
- 미용실의 회원관리와 장부관리를 효율적으로 처리할 수 있는 태블릿 기반 애플리케이션 개발
- 직관적인 UI/UX를 통한 사용자 편의성 극대화
- 데이터의 안전한 관리와 백업 기능 제공

### 1.2 주요 기능
1. 회원 관리
   - 고객 기본 정보 관리
   - 고객별 시술 이력 관리
   - 고객 방문 일정 관리
   - 고객별 선호 스타일 및 특이사항 기록

2. 장부 관리
   - 일일/월간 매출 관리
   - 시술별 매출 통계
   - 지출 관리
   - 매출/지출 리포트 생성

3. 예약 관리
   - 실시간 예약 현황 확인
   - 예약 등록/수정/취소
   - 예약 알림 기능

4. 직원 관리
   - 직원별 실적 관리
   - 근무 일정 관리
   - 급여 정산 지원

## 2. 기술 스택

### 2.1 개발 환경
- Framework: Flutter
- 개발 언어: Dart
- 타겟 플랫폼: iOS/Android 태블릿
- 최소 지원 해상도: 1080 x 720

### 2.2 백엔드 구성
- 데이터베이스: SQLite (로컬 저장)
- 백업 저장소: Firebase Storage
- 인증: Firebase Authentication

### 2.3 상태 관리
- Provider 패턴 활용
- 로컬 데이터베이스 동기화

## 3. 시스템 아키텍처

### 3.1 앱 구조
```
lib/
├── models/         # 데이터 모델
├── views/          # UI 화면
├── controllers/    # 비즈니스 로직
├── services/       # 외부 서비스 연동
├── utils/          # 유틸리티 함수
└── widgets/        # 재사용 가능한 위젯
```

### 3.2 데이터 모델
1. 회원 정보
   - 기본 정보 (이름, 연락처, 생년월일 등)
   - 시술 이력
   - 선호 스타일 정보

2. 매출/지출 정보
   - 일일 매출 기록
   - 지출 내역
   - 시술별 매출 데이터

3. 예약 정보
   - 예약 일시
   - 고객 정보
   - 시술 정보
   - 담당 직원

## 4. UI/UX 설계

### 4.1 주요 화면 구성
1. 메인 대시보드
   - 오늘의 예약 현황
   - 일일 매출 현황
   - 주요 알림

2. 회원 관리 화면
   - 회원 목록/검색
   - 회원 상세 정보
   - 시술 이력 조회

3. 장부 관리 화면
   - 일일/월간 매출 현황
   - 지출 관리
   - 통계 리포트

4. 예약 관리 화면
   - 달력 형태의 예약 현황
   - 예약 등록/수정 폼
   - 시간대별 예약 현황

### 4.2 디자인 가이드라인
- 태블릿에 최적화된 레이아웃 설계
- 직관적인 네비게이션 구조
- 터치 기반 인터랙션 최적화
- 가독성 높은 폰트 사용

## 5. 보안 및 데이터 관리

### 5.1 데이터 백업
- 주기적인 자동 백업
- 수동 백업 기능
- 백업 데이터 복원 기능

### 5.2 보안
- 사용자 인증 및 권한 관리
- 중요 데이터 암호화
- 개인정보 보호 준수

## 6. 향후 개선 계획

### 6.1 추가 기능
- 온라인 예약 시스템 연동

### 6.2 성능 최적화
