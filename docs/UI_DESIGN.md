# 타임헤어 관리 시스템 - UX/UI 디자인 가이드

## 1. 디자인 컨셉

### 컬러 팔레트

| 용도 | 색상 | HEX | 설명 |
|------|------|-----|------|
| Primary | 파란색 | `#1890ff` | Ant Design 기본, 주요 버튼/액션 |
| Success | 초록색 | `#52c41a` | 비어있음, 완료 상태 |
| Warning | 노란색 | `#faad14` | 예약됨, 주의 |
| Error | 빨간색 | `#ff4d4f` | 시술중, 긴급 |
| Background | 밝은 회색 | `#f5f5f5` | 배경색 |
| Card | 흰색 | `#ffffff` | 카드 배경 |

### 좌석 상태 색상

```
🟢 비어있음 (Available)  - #52c41a (초록)
🔴 시술중 (In Use)       - #ff4d4f (빨강)
🟡 예약됨 (Reserved)     - #faad14 (노랑)
```

## 4. 컴포넌트 스타일 가이드

### 4.1 버튼

```
Primary:    [예약 확정]     - 파란색 배경, 흰 글자
Success:    [완료]         - 초록색 배경, 흰 글자
Warning:    [시작]         - 노란색 배경, 검정 글자
Danger:     [취소]         - 빨간색 테두리, 빨간 글자
Default:    [닫기]         - 회색 테두리, 검정 글자
```

### 4.2 카드

```css
- 배경: 흰색
- 테두리: 1px solid #f0f0f0
- 그림자: 0 1px 2px rgba(0,0,0,0.03)
- 모서리: 8px
- 패딩: 16px
```

### 4.3 테이블

```css
- 헤더 배경: #fafafa
- 행 hover: #f5f5f5
- 테두리: 1px solid #f0f0f0
```

### 4.4 폰트

```css
- 제목: 18px, 굵게
- 본문: 14px
- 작은 텍스트: 12px
- 금액: 16px, 굵게
```

## 5. 인터랙션 패턴

### 5.1 좌석 카드 클릭

| 상태 | 클릭 시 동작 |
|------|-------------|
| 비어있음 | 예약 생성 모달 열기 |
| 시술중 | 시술 상세 모달 (완료 버튼) |
| 예약됨 | 예약 상세 모달 (시작/취소 버튼) |

### 5.2 알림/피드백

- 성공: 초록색 토스트 메시지 (3초)
- 오류: 빨간색 토스트 메시지 (5초)
- 확인 필요: 모달 다이얼로그

### 5.3 로딩

- 전체 페이지: 스피너
- 부분 로딩: Skeleton
